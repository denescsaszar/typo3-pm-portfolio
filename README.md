# TYPO3 Project Management Portfolio

> Praxisnahe Szenarien aus dem Agenturalltag — als Vorbereitung auf die Rolle als Projektmanager:in für digitale TYPO3-Projekte bei e-pixler GmbH.

## Worum geht es?

Dieses Repository zeigt anhand realistischer Agentur-Tickets, dass ich:
- **TYPO3-Code lesen und einordnen** kann (TypoScript, Fluid, PHP, HTML/CSS)
- **Zwischen Kunde und Entwickler uebersetzen** kann
- **Projekte strukturiert steuern** kann (Agile, Budget, Timeline)
- **Technische Entscheidungen nachvollziehen** und kommunizieren kann
